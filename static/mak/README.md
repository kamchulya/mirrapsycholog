# МАК-карты — как добавить картинки

## Куда класть файлы
Все изображения МАК-карт кладём в папку:
```
mirra/static/mak/
```

## Имена файлов (строго эти):
```
mak_01.jpg  — Старый маяк в тумане
mak_02.jpg  — Девочка с воздушными шарами
mak_03.jpg  — Дерево с корнями над обрывом
mak_04.jpg  — Две дороги в лесу
mak_05.jpg  — Птица, покидающая клетку
mak_06.jpg  — Женщина смотрит в зеркало
mak_07.jpg  — Мост через бурную реку
mak_08.jpg  — Руки, сажающие семя
mak_09.jpg  — Звёздное небо над горами
mak_10.jpg  — Закрытая дверь с ключом
mak_11.jpg  — Танцующая фигура в тени
mak_12.jpg  — Лодка без вёсел на озере
```

## Требования к изображениям
- Формат: JPG или PNG (переименуй в .jpg)
- Размер: желательно не меньше 800x600 пикселей
- Telegram показывает изображение на весь экран при нажатии

## Если картинок нет
Бот работает и без картинок — просто показывает название карты текстом.
Как только добавишь файлы — автоматически начнёт отправлять изображения.

## Медитации YouTube
В файле `handlers/main_handler.py` найди словарь `MEDITATION_YOUTUBE`
и замени ссылки `url` на нужные YouTube видео.
